Synopsis

    wc-cli deploy -r [remotehost] -p [context]  ........... deploy current web console to a remote server and publish under the given context

Options
    -o, --offline ....................... create only the zip file without remote deploy
    -r, --remote ........................ remote MOTIF host to deploy
    -p, --publish ....................... publish remotely for the given context
    -user ............................... basic authentication user name
    -passwd ............................. basic authentication user password
