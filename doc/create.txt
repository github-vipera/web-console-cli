Synopsis

  Create a new module
    wc-cli create module -n [moduleName]
    wc-cli create module -n [moduleName] -t [templateName]

  Create a new Web Console
    wc-cli create console -n [consoleName]
    wc-cli create console -n [consoleName] -t [templateName]

Options
    -n, --name .......................... the new mondule/console application name
    -t, --template ...................... the template for the new module/console application, by default 'default' is used
